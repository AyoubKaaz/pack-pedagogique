# Bibliothèque de patterns validés
Un pattern = une formulation de correction validée manuellement par
Kaazouzi Ayyoub et prête à être réutilisée dans les chapitres suivants.
## Comment alimenter ce dossier
Quand une correction est validée dans Claude Code :
  "Verse ce pattern dans patterns/nom-du-fichier.md"
Claude Code crée ou met à jour le fichier correspondant.
