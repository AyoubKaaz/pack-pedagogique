# Pattern validé — Résolution d'une équation du 1er degré
Validé dans : ch01
Skill appliqué : domaine-algebre + correction-commune
Type : SOLUTION_APPLICATION

## Structure type
```latex
On a :
\[3x - 5 = 7\]
\[\Leftrightarrow 3x = 7 + 5 \qquad \text{(on ajoute 5 aux deux membres)}\]
\[\Leftrightarrow 3x = 12\]
\[\Leftrightarrow x = 4 \qquad \text{(on divise par 3, 3 > 0)}\]
Donc $S = \{4\}$.
```
## Règles absolues
- Justification obligatoire à chaque étape entre parenthèses
- "on divise par c, c > 0" — ne jamais omettre la condition sur c
- Si c < 0 : "(on divise par c < 0, le sens de l'inégalité s'inverse)"
- Conclusion toujours : Donc $S = \{valeur\}$.  (avec point final)
- Pas de phrase ▶ \blacktriangleright (interdit en APPLICATION)
## Variantes
| Cas | Conclusion |
|-----|-----------|
| Solution unique | `Donc $S = \{valeur\}$.` |
| Aucune solution | `Donc $S = \emptyset$.` |
| Tout réel solution | `Donc $S = \R$.` |
| Inéquation | `Donc $S = [a \; ; \; +\infty[$.` |
