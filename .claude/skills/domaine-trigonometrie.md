## Valeurs remarquables — tableau de référence
| Angle | 0 | π/6 | π/4 | π/3 | π/2 |
|-------|---|-----|-----|-----|-----|
| cos   | 1 | √3/2 | √2/2 | 1/2 | 0 |
| sin   | 0 | 1/2 | √2/2 | √3/2 | 1 |
| tan   | 0 | 1/√3 | 1 | √3 | — |
## Structure type — Résoudre cos(x) = a
```latex
On a :
\[\cos(x) = a\]
\[\Leftrightarrow x = \pm \arccos(a) + 2k\pi, \quad k \in \Z\]
```
## Structure type — Formule d'addition
```latex
\[\cos(a + b) = \cos a \cos b - \sin a \sin b\]
\[\sin(a + b) = \sin a \cos b + \cos a \sin b\]
```
