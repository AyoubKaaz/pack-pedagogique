## Notations officielles (vocabulaire marocain officiel)
- Moyenne : \bar{x}  (jamais m ou μ)
- Effectif total : N
- Effectif de la classe i : n_i
- Fréquence : f_i = n_i / N
- Étendue : max - min
- Médiane : Me
- Mode : valeur de plus grand effectif
## Structure type — Calcul de la moyenne
```latex
On a :
\[\bar{x} = \frac{1}{N}\sum_{i=1}^{k} n_i x_i
= \frac{n_1 x_1 + n_2 x_2 + \ldots + n_k x_k}{N}\]
```
## Structure type — Lecture de médiane
```latex
L'effectif total est $N = \ldots$, donc $N/2 = \ldots$
La médiane $M_e$ est la valeur pour laquelle
l'effectif cumulé croissant atteint $N/2$.
D'où $M_e = \ldots$
```
## Justifications spécifiques au domaine
| Opération | Justification |
|-----------|---------------|
| Lire la médiane | `(effectif cumulé croissant)` |
| Calculer moyenne | `(somme des produits divisée par N)` |
