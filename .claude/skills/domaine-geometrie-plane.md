## Notations officielles
- Translation de vecteur \vect{u} : t_{\vect{u}}
- Rotation de centre O et d'angle θ : r_{(O,\theta)}
- Symétrie axiale d'axe (d) : s_{(d)}
- Homothétie de centre O et de rapport k : h_{(O,k)}
- Image de A par f : f(A) = A'
## Structure type — Image d'un point par translation
```latex
Soit $\vect{u}\begin{pmatrix}a\\b\end{pmatrix}$.
Si $A(x_A\,;\,y_A)$, alors son image $A'$ par $t_{\vect{u}}$ est :
\[A'\left(x_A + a \,;\, y_A + b\right)\]
```
## Structure type — Conserver les propriétés
```latex
La translation conserve :
\begin{itemize}
    \item les distances
    \item les angles orientés
    \item le parallélisme
\end{itemize}
```
