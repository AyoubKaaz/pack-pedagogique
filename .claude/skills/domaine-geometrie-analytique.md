## Notations officielles
- Équation cartésienne d'une droite : ax + by + c = 0
- Vecteur directeur : \vect{u}
- Vecteur normal : \vect{n}
- Distance : d(A, droite)
## Structure type — Équation d'une droite passant par A(x₀,y₀)
## de vecteur directeur \vect{u}(a,b)
```latex
La droite $(d)$ passant par $A(x_0\,;\,y_0)$
de vecteur directeur $\vect{u}\begin{pmatrix}a\\b\end{pmatrix}$
a pour équation :
\[b(x - x_0) - a(y - y_0) = 0\]
```
## Structure type — Distance d'un point à une droite
```latex
La distance de $A(x_0\,;\,y_0)$ à la droite
$(d) : ax + by + c = 0$ est :
\[d(A,(d)) = \frac{|ax_0 + by_0 + c|}{\sqrt{a^2 + b^2}}\]
```
